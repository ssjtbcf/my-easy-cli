# 脚手架

## 思路
- 解耦：脚手架与模板分离
    - 脚手架负责构建流程，通过命令行与用户交互，获取项目信息
    - 模板负责统一项目结构，依赖项管理
    - 脚手架检测模板的版本是否有更新，支持模板的删除与创建
